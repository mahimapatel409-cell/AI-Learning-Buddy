🤖 AI Learning Buddy

An AI-powered learning assistant designed to help students learn smarter, solve doubts, and improve their technical knowledge with the power of Artificial Intelligence.

🚀 Overview

AI Learning Buddy is an interactive AI-based application that helps students with:

📚 Learning concepts easily
💡 Getting instant explanations
🧠 Clearing doubts through AI assistance
📝 Improving study efficiency
🎯 Supporting technical interview preparation

The goal of this project is to make learning more personalized, accessible, and effective.

✨ Features

🤖 AI-powered chatbot for doubt solving
📖 Topic explanations in simple language
🔍 Quick answers for technical questions
🎓 Student-friendly learning support
⚡ Fast and interactive interface

⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https:https://github.com/mahimapatel409-cell/AI-Learning-Buddy
```

2. Move into project folder

```bash
cd AI-Learning-Buddy
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

 4. Run the application

```bash
streamlit run app.py
```
